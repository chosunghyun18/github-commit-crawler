## Docker run in dev.yml

- $ docker-compose -f docker-compose.dev.yml up -d

-d : run in backgroud
-f : choose file

## 실행 상태 확인

- $ docker ps

## 실행 상태 확인 pm2

- $ docker exec -it express_task_dev pm2 list

## 실행 중인 도커 종료

- $ docker compose down
